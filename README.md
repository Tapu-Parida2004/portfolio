# Portfolio Website

This is a personal portfolio website showcasing my skills, experience, and projects.

## Features

- **Home Page**: A welcoming introduction.
- **About Page**: A brief overview of my background and interests.
- **Contact Page**: Easy ways to get in touch.
- **Experience Page**: Highlights of my professional journey.
- **Portfolio Page**: A showcase of my projects.

## Technologies Used

- **Framework**: [Vite](https://vitejs.dev/)
- **Languages**: React, Tailwind CSS.
- **Deployment**: GitHub Pages, 

## How to Run Locally

1. Clone this repository:
   git clone https://github.com/Tapu_Parida2004/portfolio.git
2. Navigate to the project directory:
   cd your-repo-name
3. Install dependencies:
   npm install
4. Start the development server:
   npm run dev
